# Face-Detection
Face Detection system built using Python and OpenCV that detects and tracks human faces in real time. It uses pre-trained Haar Cascade classifiers to identify faces from webcam or images, making it useful for security, attendance systems, and computer vision applications.

# 🎯 Face Detection System  

A real-time Face Detection system built using Python and OpenCV. This project detects human faces from a webcam or image using pre-trained Haar Cascade classifiers, making it useful for basic computer vision applications.

---

## 📌 Features  
- 🎥 Real-time face detection using webcam  
- 🖼️ Face detection in images  
- ⚡ Fast and efficient using OpenCV  
- 🧠 Uses Haar Cascade Classifier  
- 📦 Easy to set up and run  

---

## 🛠️ Tech Stack  
- Python  
- OpenCV  
- NumPy  

---

## ⚙️ How It Works  
1. Capture video from webcam or load an image  
2. Convert frame to grayscale  
3. Apply Haar Cascade classifier  
4. Detect faces and draw bounding boxes  
5. Display output in real time

## 🚀 Installation  

```bash
# Clone the repository
git clone https://github.com/your-username/face-detection.git

# Navigate to the folder
cd face-detection

# Install dependencies
pip install opencv-python numpy
```

---

## ▶️ Usage  

```bash
python face_detection.py
```

Make sure your webcam is connected and working.
